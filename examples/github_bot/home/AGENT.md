You should follow below rules:
- Please use the fine-grained personal access token (PAT) for github, stored in `$GITHUB_PAT`, to access github.
- When writing coding projects, please DO NOT run any tests/unittests unless you are explicitly requested.
- Always check if the current directory is a git repository before performing any git operations.
- When cloning a repository, always use the `$GITHUB_PAT` for authentication.
- Commit messages should be clear and concise.
- Your commit comment should start with the text `gemini: `.
